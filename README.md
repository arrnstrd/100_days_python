# 🐍 100 Days of Python

> *From zero to builder — one day at a time.*

A structured, self-directed journey through Python fundamentals, mini-projects, and data structures & algorithms. The goal isn't just to learn syntax — it's to **think like a programmer**.

---

## 📌 What This Is

| | |
|---|---|
| **Duration** | 100 days |
| **Stack** | Python, JSON, CSV, Matplotlib (optional) |
| **Focus** | Fundamentals → Projects → DSA |
| **End Goal** | A portfolio-worthy CLI application + solid problem-solving foundation |

---

```
Phase 1 ──────────────── Phase 2 ──────────── Phase 3

```
